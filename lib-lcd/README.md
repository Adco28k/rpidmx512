- LCD Module TC1602A / PCF8574T I2C ([https://www.hobbyelectronica.nl/product/1602-lcd-i2c-blauw-backlight/](https://www.hobbyelectronica.nl/product/1602-lcd-i2c-blauw-backlight/))
